# 🚀 FlowBoard

**FlowBoard** — это веб-приложение для управления проектами в стиле Kanban с реальным временем, чатом и совместной работой.

Создан как лёгкая альтернатива Trello / Notion для команд, которым нужна простота и скорость.

---

## ⚡ Возможности

* 📋 Kanban-доски (колонки и карточки)
* 🔄 Drag & Drop (перетаскивание задач)
* 💬 Чат проекта в реальном времени
* 👥 Совместная работа (через ссылку)
* 🏷️ Теги (labels) для задач
* 🖼️ Обложки карточек (изображения)
* 📝 Markdown в описании
* 🔐 Авторизация через Google
* ☁️ Синхронизация через Firebase
* 🌙 Тёмная тема

---

## 🧠 Как это работает

* Все данные хранятся в **Firebase Firestore**
* Авторизация через **Firebase Auth (Google)**
* UI построен на **TailwindCSS**
* Drag & Drop через **SortableJS**
* Markdown через **Marked.js**

---

## ⚙️ Установка

1. Клонируй проект:

```
git clone https://github.com/your-username/flowboard.git
```

2. Открой `config.js` и вставь свои ключи Firebase 

```js
const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  projectId: "...",
};
```

3. Запусти локально (например через Live Server)
